# boostCX

**Platform-first customer experience software.**

boostCX builds shared platform services that power reporting, AI insights, and communications for multi-tenant customer experience programs — delivered through configuration, not custom code.

---

### How we work

- Issues and pull requests follow structured templates with required quality gates
- All changes go through pull requests with CI checks and peer review
- AI-assisted development is encouraged; engineers own the output
- Security vulnerabilities are reported privately per our [security policy](https://github.com/boostCX/.github/blob/main/SECURITY.md)
